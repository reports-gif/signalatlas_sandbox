# Changelog

## 1.0.0 — 2026-03-04

Initial release.

- Live theme editor accessible at Administration → System → Custom Theme
- 34 colour controls covering all major Matomo UI variables
- Automatic harmonious palette generation from a single primary colour (HSL colour theory)
- Background image upload with display mode, overlay opacity, and blur controls
- Typography: 11 curated font stacks plus custom font upload (WOFF2/WOFF/TTF/OTF)
- Shape roundness: 5 presets applied consistently across all UI elements
- Live colour preview before saving
- One-click reset to Matomo defaults
- Security: super-admin only, hex validation, MIME/magic-byte file validation, SVG blocked, authenticated file proxy, CSRF protection
